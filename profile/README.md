> **超级龙虾 (WonderClaw)** 是由 [超级简历 WonderCV](https://www.wondercv.com) 发起的 AI 开源实验室。我们致力于通过 **AI Skills** 为你的 AI Agent 装上最强力的"龙虾之螯"，精准、有力、专业地抓取每一个职业机会。

---

## 💡 品牌理念

在 AI 时代，通用的模型需要专业的"技能模块"才能处理复杂的职场任务。**超级龙虾** 专注于将 **超级简历** 积累的专业求职方法论解构为可插拔的 **AI Skills**，让每一位开发者和求职者都能在自己最熟悉的编辑器或终端中（如 Claude Code, Codex CLI），瞬间获得顶尖 HR 的实战能力。

---

## 🚀 核心开源项目

### [clawcv](https://github.com/WonderClaw/clawcv)
**ClawCV** 是由**超级龙虾**推出的全功能 AI 求职技能集入口。
- 🦞 **龙虾一键装配**：支持通过 `npx clawcv` 一键安装全套求职 Skill。
- 📖 **配置指南**：提供针对 Claude Code、Codex CLI 等主流 AI 终端的 API Key 配置手册。
- 🛠️ **全能 Skill 库**：内置简历诊断、智能改写、JD 匹配、PDF 导出、AI 导师、账号管理等 6 大核心模块。

---

## 🛠️ 核心技能优势

就像强有力的龙虾之螯，**超级龙虾** 赋予 AI Agent 以下专业深度：

*   **专业表达模型**：基于 **160 万字** 行业语料库，精准抓取经历中的闪光点，并转化为 **STAR 原则** 描述。
*   **ATS 深度穿透**：内置岗位关键词密度检测与兼容性校准，确保你的简历能被企业筛选系统精准识别。
*   **HR 视角纠错**：像显微镜一样发现简历中隐藏的 **20+ 类** 逻辑与格式硬伤。
*   **Skill-First**：采用模块化的 Skill 设计，支持自然语言触发或 `/` 指令精准调度。

## 🌟 3 秒钟，激活你的超级龙虾

1. **领取 Key**：前往 [wondercv.com/clawcv](https://wondercv.com/clawcv) 注册并获取 API Key。
2. **装配 Skills**：运行 `npx clawcv` 完成超级龙虾的技能装载。
3. **注入能量**：设置 `WONDERCV_API_KEY` 环境变量。
4. **即刻开工**：对 AI 说 **"帮我分析这份简历"** 或 **"把我的工作经历改得更像产品经理"**。
